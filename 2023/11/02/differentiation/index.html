导数

导数定义

对于一个函数 f(x)，如果在 x=a的邻域内，函数 f(x) 的增量与自变量 x的增量之比在 x 趋近于a时的极限存在，那么称函数 f(x) 在 x=a 处可导，并称这个极限值为函数 f(x)在x=a 处的导数，记作 f'(a) 或 df(x)/dx|x=a

导数的几个性质

线性性：导数具有线性性质，即对于任意常数 c1 和 c2 以及函数 f(x) 和 g(x)，有：
(c1 * f(x) + c2 * g(x))' = c1 * f'(x) + c2 * g'(x)

乘法法则：导数具有乘法法则，即对于函数 f(x) 和 g(x)，有：

(f(x) * g(x))' = f'(x) * g(x) + f(x) * g'(x)

除法法则：导数具有除法法则，即对于函数 f(x) 和 g(x)，有：

(当 g(x) ≠ 0 时,f(x) / g(x))' = (f'(x) * g(x) - f(x) * g'(x)) / g(x)^2$

链式法则：导数具有链式法则，即对于复合函数 f(g(x))，有：
(f(g(x)))' = f'(g(x)) * g'(x)

导数与函数的增减性：如果函数 f'(x) > 0（在某一区间内），则函数 f(x) 在该区间内单调递增；如果 f'(x) < 0，那么函数 f(x) 在该区间内单调递减。

导数与函数的极值：函数 f(x) 在某点 x=a 处达到极大值或极小值时，如果在该点处可导，则 f'(a) = 0。需要注意的是，f'(a) = 0 并不一定意味着函数在该点达到极值，还需要进行进一步的判断。

一元微分

微分意义

假设我们有一个函数 y = f(x)，当 x发生微小变化时，我们希望知道y的微小变化。为了解决这个问题，我们引入了微分的概念。

我们用 dy 表示函数的微小变化，用 dx 表示自变量的微小变化。这里的 dx 和 dy 都被认为是无穷小量，即非常小的变化。

一元微分形式的不变性

我们可以用导数来计算微分：

dy = f'(x) · dx

微分中值定理

罗尔定理

若函数 f满足如下条件：

（1） f在闭区间 [a,b] 上连续；

（2） f 在开区间 （a，b）上可导；

（3） f(a) = f(b)

则在 （a，b）上至少存在一点 ɳ ，使得 f'(ɳ)=0

拉格朗日中值定理

若函数 f满足如下条件：

（1） f在闭区间 [a,b] 上连续；

（2） f 在开区间 （a，b）上可导；

则在 （a，b）上至少存在一点 ɳ ，使得


f'(ɳ) =f(b)-f(a) \ b-a

柯西中值定理

若函数 f,g满足如下条件：

（1） f,g在[a,b] 上连续；

（2） f,g 在[a，b]上可导；

（3） 两个导数不同时为0

（4） g(a) 不等于 g(b)
则在 （a，b）上至少存在一点 ɳ ，使得


f'(ɳ) \ g'(ɳ)=f(b)-f(a) \ g(b)-g(a)
